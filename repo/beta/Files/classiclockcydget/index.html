Requires iOS 7+ (definitely!)
Compatible with all iPhones, iPads and iPod touches!

NOTICE: This is a beta version of ClassicLockCydget, it may contain bad bugs. If you installed a previous version from the BigBoss repo and you don't want to update, you can hide this in the menu above.
ClassicLock Cydget is a free iOS 6 Lockscreen Cydget on iOS 7+
It is very easy expandable so you can have many cool widgets and extensions on your fake-iOS6-Lockscreen

---Setup---
First install this package and the dependencies. Go into settings app -> Cydget -> ClassicLock. Select and put it on top of the list.
Respring to apply. Now you should see the good old iOS 6 Lockscreen just like you know it from the ancient days :)
